Configuração do SisPharma 1.0 ===========================================================

1. Navegue até a pasta 'SisPharma software' em 'Arquivos de Programas';

2. Instale o 'jre';

3. Instale o 'mysql' e configure a senha para 'admin';

4. No 'SQLyog' import o banco de dados 'bd' que está na pasta bdSisPharma;

5. Para entrar no sistema login: admin
                          senha: admin
